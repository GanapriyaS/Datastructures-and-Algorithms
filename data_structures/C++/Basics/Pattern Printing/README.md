# Pattern Printing

This directory contains C++ programs for printing various stars, numbers, and alphabetical patterns to standard output.

## Programs Index

- [2_stars.cpp](./2_stars.cpp) - Print 2-Stars pattern.
- [m_stars.cpp](./m_stars.cpp) - Print M-Stars pattern.
- [square.cpp](./square.cpp) - Print a filled square pattern.
- [hollow_square.cpp](./hollow_square.cpp) - Print a hollow square pattern.
- [hollow_rectangle.cpp](./hollow_rectangle.cpp) - Print a hollow rectangle pattern.
- [pyramid.cpp](./pyramid.cpp) - Print a star pyramid.
- [inverse_pyramid.cpp](./inverse_pyramid.cpp) - Print an inverse star pyramid.
- [numbered_triangle.cpp](./numbered_triangle.cpp) - Print a numbered triangle pattern.
- [binary_pyramid.cpp](./binary_pyramid.cpp) - Print a binary (0-1) pyramid.
- [vertical_triangle.cpp](./vertical_triangle.cpp) - Print a vertical triangle star pattern.
- [hollow_vertical_triangle.cpp](./hollow_vertical_triangle.cpp) - Print a hollow vertical triangle star pattern.
- [triangle.cpp](./triangle.cpp) - Print a basic triangle star pattern.
- [hollow_triangle.cpp](./hollow_triangle.cpp) - Print a hollow triangle star pattern.
- [inverted_hollow_triangle.cpp](./inverted_hollow_triangle.cpp) - Print an inverted hollow triangle star pattern.
- [diamond.cpp](./diamond.cpp) - Print a star diamond.
- [hollow_diamond.cpp](./hollow_diamond.cpp) - Print a hollow star diamond.
- [crown.cpp](./crown.cpp) - Print a crown pattern.
- [butterfly.cpp](./butterfly.cpp) - Print a butterfly pattern.
- [inverted_diamond.cpp](./inverted_diamond.cpp) - Print an inverted star diamond.
