# Loops

This directory contains programming exercises focusing on iteration control structures (loops like `for`, `while`, and `do-while`) in C++.

## Programs Index

- [print_1_to_n.cpp](./print_1_to_n.cpp) - Print numbers from 1 to N.
- [print_n_to_1.cpp](./print_n_to_1.cpp) - Print numbers in reverse from N to 1.
- [print_all_even_numbers_from_1_to_n.cpp](./print_all_even_numbers_from_1_to_n.cpp) - Print all even numbers from 1 to N.
- [print_from_l_to_r.cpp](./print_from_l_to_r.cpp) - Print all numbers in the range [L, R].
- [print_all_uppercase_alphabets.cpp](./print_all_uppercase_alphabets.cpp) - Print all uppercase English alphabets.
- [print_table_of_n.cpp](./print_table_of_n.cpp) - Print multiplication table of N.
- [count_numbers.cpp](./count_numbers.cpp) - Count specific subsets of numbers under conditions.
- [sum_of_first_n_natural_numbers.cpp](./sum_of_first_n_natural_numbers.cpp) - Calculate the sum of the first N natural numbers.
- [factorial.cpp](./factorial.cpp) - Compute the factorial of a given number.
- [x_power_n.cpp](./x_power_n.cpp) - Calculate $x$ raised to the power $n$ ($x^n$).
- [print_number_in_reverse.cpp](./print_number_in_reverse.cpp) - Print individual digits of a number in reverse.
- [sum_of_digits.cpp](./sum_of_digits.cpp) - Calculate the sum of the digits of a number.
- [reverse_number_and_store_in_a_variable.cpp](./reverse_number_and_store_in_a_variable.cpp) - Reverse a number and store the result in a variable.
- [palindrome.cpp](./palindrome.cpp) - Check if a number is a palindrome.
