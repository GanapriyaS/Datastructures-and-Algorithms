# Graph Algorithms

This directory contains key graph representations and traversal/minimum-spanning-tree algorithms.

## Programs Index

- [breadth first search.cpp](./breadth%20first%20search.cpp) - Breadth First Search (BFS) graph traversal.
- [depth first search.cpp](./depth%20first%20search.cpp) - Depth First Search (DFS) graph traversal.
- [kruskal's algorithm.cpp](./kruskal%27s%20algorithm.cpp) - Kruskal's Algorithm to find Minimum Spanning Tree (MST).
- [prim algorithm.cpp](./prim%20algorithm.cpp) - Prim's Algorithm to find Minimum Spanning Tree (MST).
