# Classes

This directory contains foundational Object-Oriented Programming (OOP) concepts, class structures, constructors, destructors, member overloading, and static variables in C++.

## Programs Index

- [copy_constructor.cpp](./copy_constructor.cpp) - Demonstrates OOP Copy Constructors.
- [default_constructor.cpp](./default_constructor.cpp) - Demonstrates OOP Default Constructors.
- [default_fun.cpp](./default_fun.cpp) - Shows usage of default arguments in functions.
- [destructor.cpp](./destructor.cpp) - Shows class destructors and memory cleanup.
- [friendfun.cpp](./friendfun.cpp) - Friend functions sharing class private members.
- [friendfun1class.cpp](./friendfun1class.cpp) - A class-level friend function example.
- [funoverloading.cpp](./funoverloading.cpp) - Standard function overloading demonstration.
- [inline.cpp](./inline.cpp) - Inline functions for optimization.
- [macro.cpp](./macro.cpp) - Macro preprocessors in C++.
- [overloading.cpp](./overloading.cpp) - Advanced operator/function overloading.
- [power.cpp](./power.cpp) - Program to calculate the power of a number.
- [staticmem.cpp](./staticmem.cpp) - Static class members.
- [staticvar.cpp](./staticvar.cpp) - Static variables in C++.
- [student.cpp](./student.cpp) - A basic student class structure.
