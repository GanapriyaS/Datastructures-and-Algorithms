# Introductory Problems

This directory contains introductory level programming exercises for getting started with C++ syntax, operators, basic input/output, conditionals, and loops.

## Programs Index

- [hello_world.cpp](./hello_world.cpp) - Hello World program.
- [print_first_5_alphabets.cpp](./print_first_5_alphabets.cpp) - Print the first 5 alphabets.
- [triangle.cpp](./triangle.cpp) - Area or drawing of a triangle.
- [print_z.cpp](./print_z.cpp) - Print character pattern Z.
- [print_table_of_5.cpp](./print_table_of_5.cpp) - Print multiplication table of 5.
- [rectangle.cpp](./rectangle.cpp) - Area or properties of a rectangle.
- [print_table_of_n.cpp](./print_table_of_n.cpp) - Print multiplication table of N.
- [calculator.cpp](./calculator.cpp) - A basic arithmetic calculator.
- [calculator_ii.cpp](./calculator_ii.cpp) - Calculator version II with additional operations/logic.
- [add_last_digits.cpp](./add_last_digits.cpp) - Add last digits of two numbers.
- [even_or_odd.cpp](./even_or_odd.cpp) - Check if a number is even or odd.
- [factor.cpp](./factor.cpp) - Find factors of a number.
- [multiple.cpp](./multiple.cpp) - Check if one number is a multiple of another.
- [pass_or_fail.cpp](./pass_or_fail.cpp) - Check student pass/fail status based on marks.
- [max_and_min_of_2_numbers.cpp](./max_and_min_of_2_numbers.cpp) - Find maximum and minimum of two numbers.
- [max_and_min_of_3_numbers.cpp](./max_and_min_of_3_numbers.cpp) - Find maximum and minimum of three numbers.
- [student_performance_evaluation.cpp](./student_performance_evaluation.cpp) - Evaluate student performance grading.
- [find_the_location_point.cpp](./find_the_location_point.cpp) - Determine coordinate location points.
