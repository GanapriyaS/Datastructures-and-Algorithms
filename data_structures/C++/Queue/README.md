# Queue

This directory contains queue implementation methods and job scheduling algorithms using queues.

## Programs Index

- [circular queue.cpp](./circular%20queue.cpp) - Circular Queue operations.
- [fcfs.cpp](./fcfs.cpp) - First-Come, First-Served (FCFS) CPU scheduling algorithm.
- [queue operations using array.cpp](./queue%20operations%20using%20array.cpp) - Queue operations implemented using arrays.
- [queue operations using linkedlist.cpp](./queue%20operations%20using%20linkedlist.cpp) - Queue operations implemented using linked lists.
- [round robin scheduling.cpp](./round%20robin%20scheduling.cpp) - Round Robin scheduling simulation using queues.
